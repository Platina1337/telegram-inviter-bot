# Подпись под пересланным постом — как пользоваться

## Что это

В режиме **копирования** постов (когда «Нативная пересылка» выключена) можно включать **подпись** — в конец каждого пересланного поста будет добавляться блок с ссылками и вашим текстом.

## Как включить

1. В настройках пересылки выключите **«Нативная пересылка»** (должно быть «❌ Выкл»).
2. Нажмите **«✍️ Добавлять подпись»**, чтобы включить (станет ✅).
3. Нажмите **«✏️ Настроить подпись»** — откроется меню настройки подписи.

## Что можно настроить

В меню **«Настройка подписи»**:

- **Ссылка на пост** — прямая ссылка на конкретное сообщение (например, `t.me/c/123/456`).
- **Ссылка на источник (канал)** — ссылка на канал/группу без номера сообщения.
- **Ссылка на автора** — ссылка на пользователя или канал, отправивший пост.

Каждый пункт можно включить (✅) или выключить (❌). Можно оставить только один вид ссылки или любую комбинацию.

Для каждого типа ссылки задаётся **свой текст** (приписка перед ссылкой):

- **Текст для поста** — подпись перед прямой ссылкой на сообщение. По умолчанию: `Ссылка на пост`.
- **Текст для источника** — подпись перед ссылкой на канал/группу. По умолчанию: `Источник`.
- **Текст для автора** — подпись перед ссылкой на отправителя. По умолчанию: `Обращаться по объявлению сюда:`.

Все три текста можно заменить на свои в меню настройки подписи.

После изменений нажмите **«✅ Готово»**, чтобы вернуться к основным настройкам и запустить задачу.

## Пример результата

При включённых «Ссылка на пост», «Ссылка на источник» и «Ссылка на автора» с текстами по умолчанию в конце поста будет что-то вроде:

```
Ссылка на пост: Testgroup (https://t.me/testgroup_parser/476)
Источник: Testgroup (https://t.me/testgroup_parser)
Обращаться по объявлению сюда: Miaso Ebanoe (https://t.me/armianin12)
```

Каждый из трёх текстов можно заменить в настройках подписи на свой.

## Важно

- Подпись доступна только при **копировании** постов (нативная пересылка должна быть выключена).
- Работает и в **парсинге**, и в **мониторинге** постов.
- Если у канала/группы или автора нет публичного username, ссылка на них может не строиться (такие строки в подписи не появятся).
